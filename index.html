★ The Wikipedia Navigation Game ★

Can you find the path?

Start on one Wikipedia article. Reach another — using only the links inside each page. The fewer clicks, the better.

How to Play
01

Pick a Difficulty

Choose Easy, Medium, or Hard. Harder modes use more obscure Wikipedia articles.

02

Get Your Pages

You're given a random Start page and a Target page to reach.

03

Click Links

Navigate by clicking links inside each article. Every click counts as one move.

04

Reach the Target

Land on the target article and win. Fewer clicks = higher rating.
